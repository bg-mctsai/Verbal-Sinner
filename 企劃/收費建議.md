《嘴砲王者》平台發行與收費策略建議

一、 Steam vs. Web 平台特性對比

| 特性        | Steam (買斷制/付費下載)                                   | Web (網頁版/免費 + 內購)                             |
| ----------- | --------------------------------------------------------- | ---------------------------------------------------- |
| 初期收入    | 較高（每份賣 $5~15 USD）。                                | 較低（依賴廣告與小額課金）。                         |
| AI 成本風險 | 極高。玩家付費一次後可能進行無限對話，導致 API 費用透支。 | 可控。透過能量系統限制對話次數，確保成本與收入掛鉤。 |
| 玩家預期    | 預期高品質、完整內容、無廣告。                            | 接受廣告、體力限制與零碎遊玩。                       |
| 更新頻率    | 需穩定版本，大改版較慢。                                  | 適合快速迭代，每日更新對話邏輯。                     |

二、 為什麼「純 Steam 賣斷制」對 AI 遊戲不安全？

成本透支風險： Steam 玩家習慣「買斷後無限玩」。若一位玩家沉迷於與 NPC 聊天，產生的 Gemini API 費用可能在幾天內就吃掉你的售價利潤。

退款機制： Steam 的 2 小時退款政策。玩家可能在 2 小時內瘋狂刷 AI 語音，然後退款，留下帳單給你支付。

金鑰管理： 在本地客戶端運行 API Key 極不安全，容易被駭客提取後盜刷。

三、 建議的發行階段策略

第一階段：Web 版先行 (MVP 測試)

目標： 收集玩家對話數據，優化「語義快取庫」。

收費：免費遊玩 + 激勵影片廣告 + 少量內購（體力/道具）。

優勢： * 透過廣告收入補貼 API 支出。

測試玩家對不同年齡劇本的喜好。

建立快取庫後，能大幅降低未來進入 Steam 時的 AI 成本。

第二階段：Steam 啟動 (收費/免費雙軌)

如果您堅持上架 Steam，建議採取以下模式：

模式 A：免費下載 + 點數儲存 (Free to Play)

遊戲本體免費，但「嘴砲點數」或「AI 額度」需要透過內購或觀看廣告（Steam 不支援廣告，故需轉為內購）。

模式 B：賣斷制 + 內建額度 (Hybrid)

售價 $9.99 USD，內含「50,000 字 AI 對話額度」。

用完後，玩家需購買「重生能量包」或等待每日贈送額度。

四、 如何在 Steam 降低 AI 費用？

內建快取 (Offline Mode)： 將 Web 版中 80% 玩家常說的話與對應的 AI 回覆轉為「本地文本庫」。只有當玩家說出極其特殊的內容時，才調用雲端 API。

限制語音： Steam 版的基礎對話不提供語音，只有重要劇情（Boss 戰）才提供 AI 即時生成的語音。

自備 API Key： （進階做法）允許高階玩家在設定中輸入自己的 Gemini API Key，這樣你就不需要負擔他們的成本（適合硬核玩家）。

五、 結論：最適合的起手式

不建議剛開始就直接在 Steam 賣斷。

推薦做法：

製作一個功能完整的 Web 網頁版。

使用 Firebase Hosting 部署，並在社交媒體（Threads, TikTok）擴散。

透過**「能量系統 + 廣告」**實現損益平衡。

當遊戲熱度與 AI 快取庫成熟後，再將精華內容打包成 Steam 版本 以獲得一波爆發性收入。

「先在網頁版養活 AI 帳單，再到 Steam 賺取品牌利潤。」
